How it works

  • Tap anywhere on the timer:
    • 1× = Start / Pause / Resume
    • 2× = Reset current phase to start
    • 3× = Next phase (Work↔Rest), rep +1 when leaving Rest
  • Blue ball (left): tap to hide/show controls.
  • Swipe left on main screen to open this page.
  • Swipe right / up / down to return to the main timer.
  • Rep (top-right): tap +1; long-press resets to 1.
  • Defaults: Work 1:00, Rest 0:20.
